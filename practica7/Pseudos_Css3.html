::first-line

Se utiliza para añadir un estilo especial a la primera línea de un texto.

p::first-line {
                    color: #ff0000;
                    font-variant: small-caps;
                    text-align: left;
                }

Coincide con la porción de un elemento que es seleccionado por un usuario.
